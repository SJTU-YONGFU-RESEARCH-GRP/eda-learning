## Outcomes

After this module you can: **Obtain a multiway partition by recursively bipartitioning parts until k parts are reached.**

## Two tracks (pick one or both)

### Track A — Implement (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and implement **recursive bisection to k parts** on the tiny graphs.
2. Complete [CHECKLIST.md](CHECKLIST.md) with metrics (cutsize, balance, objective).
3. Optional self-check: `./scripts/module.sh 02-07 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/recursive-bisection/index.html](http://127.0.0.1:8080/tools/recursive-bisection/index.html)
2. Tools shelf: open `recursive-bisection` from the platform tools index
3. Load the **starter graph**, run the algorithm, inspect metrics.
4. Check off Track B items in [CHECKLIST.md](CHECKLIST.md).
