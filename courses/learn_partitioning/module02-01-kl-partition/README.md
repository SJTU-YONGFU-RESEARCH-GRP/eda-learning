## Outcomes

After this module you can: **Run full KL pair-swap bipartition from a seed cut with gains, locking, and rollback to the best prefix.**

## Two tracks (pick one or both)

### Track A — Implement (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and implement **Kernighan–Lin (KL) bipartition** on the tiny graphs.
2. Complete [CHECKLIST.md](CHECKLIST.md) with metrics (cutsize, balance, objective).
3. Optional self-check: `./scripts/module.sh 02-01 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/kl-partition/index.html](http://127.0.0.1:8080/tools/kl-partition/index.html)
2. Tools shelf: open `kl-partition` from the platform tools index
3. Load the **starter graph**, run the algorithm, inspect metrics.
4. Check off Track B items in [CHECKLIST.md](CHECKLIST.md).
