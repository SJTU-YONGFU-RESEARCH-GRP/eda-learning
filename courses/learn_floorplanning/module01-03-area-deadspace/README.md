## Outcomes

After this module you can: **Compute outline area, sum of module areas, packing density, and deadspace = outlineArea − Σ module areas.**

2. Complete [CHECKLIST.md](CHECKLIST.md) with legality, area, density, and deadspace (plus lab-specific metrics).
3. Optional self-check: `./scripts/module.sh 01-03 --check` (from course root).
