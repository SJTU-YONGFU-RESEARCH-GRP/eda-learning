## Outcomes

After this module you can: **Fill row slots with Tetris-style placement order while respecting widths and legality.**

## Two tracks (pick one or both)

### Track A — Implement (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and implement **Tetris-style row slot packing** on `examples/tiny_legal.json`.
2. Complete [CHECKLIST.md](CHECKLIST.md) with legality, displacement, and HPWL when relevant.
3. Optional self-check: `./scripts/module.sh 02-07 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/tetris-row-pack/](http://127.0.0.1:8080/tools/tetris-row-pack/)
2. Live: [https://universal-verification-methodology.github.io/learning/tools/tetris-row-pack/](https://universal-verification-methodology.github.io/learning/tools/tetris-row-pack/)
3. Tools shelf: open `tetris-row-pack` from the platform tools index
4. Load the **starter legalization**, run the algorithm, inspect legality and metrics.
5. Check off Track B items in [CHECKLIST.md](CHECKLIST.md).
