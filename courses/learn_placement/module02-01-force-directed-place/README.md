## Two tracks (pick one or both)

### Track A — Implement (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and implement **force-directed (spring) placement** on the tiny placement.
2. Complete [CHECKLIST.md](CHECKLIST.md) with metrics (HPWL, density / overflow when relevant).
3. Optional self-check: `./scripts/module.sh 02-01 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/force-directed-place/index.html](http://127.0.0.1:8080/tools/force-directed-place/index.html)
2. Tools shelf: open `force-directed-place` from the platform tools index
3. Load the **starter placement**, run the algorithm, inspect metrics.
4. Check off Track B items in [CHECKLIST.md](CHECKLIST.md).
