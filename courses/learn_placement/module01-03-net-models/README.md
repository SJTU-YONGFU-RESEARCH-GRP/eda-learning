After this module you can: **Compare bbox, clique, and star wirelength models on the same multi-pin net.**

## Two tracks (pick one or both)

### Track A — Implement (hands-on)

1. Open [EXAMPLES.md](EXAMPLES.md) and implement **bbox / clique / star net wirelength models** on the tiny placement.
2. Complete [CHECKLIST.md](CHECKLIST.md) with metrics (HPWL, density / overflow when relevant).
3. Optional self-check: `./scripts/module.sh 01-03 --check` (from course root).

### Track B — Browser lab (online)

1. Local: [http://127.0.0.1:8080/tools/net-models/index.html](http://127.0.0.1:8080/tools/net-models/index.html)
2. Tools shelf: open `net-models` from the platform tools index
3. Load the **starter placement**, run the algorithm, inspect metrics.
4. Check off Track B items in [CHECKLIST.md](CHECKLIST.md).
